HTML Assignment: Build a Personal Portfolio Webpage

Objective:

Create a personal portfolio webpage using HTML and basic CSS. This webpage will display a brief introduction, skills, projects, and contact information.


Use HTML and Inline style only.

Instruction:

  1. Create an HTML file (e.g., index.html)

    that contains the following sections:

    • Header Section:

      • The title of the portfolio (e.g., "John Doe – Web Developer")

      • A navigation bar with links to Home, About, Skills, Projects, and Contact

    • Hero Section:

      • A large profile image.

      • A short welcome message (e.g., "Hello! I am John Doe, a passionate Web Developer").

      • A short description about yourself.

    • About Me Section:

      • A paragraph describing yourself, including your education, experience, and interests.

    • Skills Section:

      • An unordered list(<ul>) showcasing at least five technical skills (e.g., HTML, CSS, JavaScript, React, Node.js).

      • Use bold text for skill names.

    • Projects Section:

      • A table (<table>) listing at least three projects, with columns:

        • Project Name

        • Description

        • Technologies Used

      • Each project should be linked to its GitHub repository or a demo.

    • Contact Section:

      • A contact form with the following fields:

        • Full Name (<input type="text">)

        • Email (⁢input type="email">)

        • Message (⁢textarea>)

        • Submit button

    • Footer Section:

      • A copyright notice (e.g., "© 2024 John Doe. All rights reserved.")

      • Social media links using icons.

  2. Apply Basic CSS (style.css)

    • Use external CSS for styling

    • Styling requirements:

      • The background of the page should have a light color or an image.

      • The navigation bar should be horizontally aligned with hover effects.

      • The hero sectionshould have a large profile picture and centered text.

      • The skills list should be neatly styled with proper spacing.

      • The projects table should have borders, padding, and hover effects.

      • The contact form should have proper spacing and a styled submit button.

      • The footer should be centered and have small social media icons.

  3. Bonus Challenges (Optional but Recommended)

    • Add smooth scrolling for navigation links.

    • Use CSS Grid or Flexbox for layout structuring.

    • Add hover effects for project links and buttons.
